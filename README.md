# Installation & Setup

Follow these steps to run the project locally:

## Clone the repository
git clone https://github.com/AmanMomin2207/yc-directory
cd yc-directory

## Install dependencies
npm install

## Start the development server
npm run dev

## Your app will run on:

http://localhost:5173
